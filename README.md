# Installation

### To install the package, open R and type:


``` r
install.packages("devtools")
```

### Then, you can install TaiwanSSP585:


``` r
devtools::install_github("chihlinwei/TaiwanSSP585", dependencies = TRUE)
library(TaiwanSSP585)
```
